# 🌱 AgriSense AI - Crop Health Monitoring System

An end-to-end **MLOps + Remote Sensing project** that uses **satellite NDVI data** and a **machine learning model** to monitor crop health and visualize results on an interactive map.

---

## 🚀 Live Demo
👉 [Streamlit App](https://your-streamlit-link-here.streamlit.app)

---

## 📌 Project Overview

AgriSense AI helps in:
- 🌍 Analyzing crop health using NDVI (Normalized Difference Vegetation Index)
- 🛰️ Visualizing satellite-based farm data on interactive maps
- 🤖 Predicting crop health using a trained ML model
- 📊 Providing a simple dashboard for real-time insights

---

## 🧠 Tech Stack

- Python 🐍
- Streamlit (Frontend Dashboard)
- Scikit-learn (ML Model)
- Folium (Geospatial Visualization)
- Pandas & NumPy (Data Processing)
- Google Earth Engine (Optional Satellite Data Source)

## ⚙️ Features

- 🌾 Crop health classification (Healthy / Moderate / Poor)
- 🛰️ Satellite-based NDVI visualization
- 🗺️ Interactive farm map with markers
- 📊 Real-time prediction using ML model
- 🎛️ User-friendly dashboard (Streamlit)

---

## 📊 How It Works

1. Satellite NDVI data is collected or simulated
2. Data is processed into features
3. ML model predicts crop health
4. Results are visualized on interactive map
